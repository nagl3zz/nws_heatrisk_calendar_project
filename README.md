# 2026 HeatRisk Calendar

This project generates color-coded **HeatRisk calendars** for any city in the NWS HeatRisk dataset,
then serves them as a simple static website using GitHub Pages.

## Project structure

```text
nws-heatrisk-calendar/
├── data/                 # Place your unzipped "NWS Data" folder here (not committed)
├── src/
│   └── heatrisk_calendar_color.py
├── docs/                 # GitHub Pages site lives here
│   ├── index.html
│   ├── style.css
│   ├── script.js
│   └── img/              # Calendar PNGs generated by the script
├── requirements.txt
├── .gitignore
└── README.md
```

## Usage

1. Create a virtual environment (optional but recommended).
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Unzip your `NWS Data.zip` so that you have:

   ```text
   data/NWS Data/<CityName>/*.csv
   ```

4. Run the script to generate calendars:

   ```bash
   python src/heatrisk_calendar_color.py
   ```

   - Choose a city from the list.
   - Choose whether to generate a single month or all 12 months.
   - PNGs will be saved directly into `docs/img/` as:

     ```text
     <city>_<month>_2026.png
     ```

     For example:

     - `yuma_june_2026.png`
     - `ajo_june_2026.png`

5. Commit and push to GitHub, then enable **GitHub Pages** with the `docs/` folder as the source.

Once GitHub Pages is enabled, your site will be available at:

```text
https://<your-username>.github.io/<your-repo-name>/
```

The site lets users pick a **city** and **month** and shows the corresponding HeatRisk calendar image.
