# HeatRisk Calendar – Year View

This project generates **12‑month HeatRisk calendar grids** for each city and year
in the NWS HeatRisk dataset, plus a single **Average** view (shown using the 2026
calendar layout).

- Each output image shows **all 12 months at once** in a 3×4 grid.
- The website lets you select **City** and **Year**, plus an **Average** option.
- Colors follow the provided scale:

  - Green:  #d9f4cd  (0 – None)
  - Yellow: #ffde59  (1 – Minor)
  - Orange: #fd933c  (2 – Moderate)
  - Red:    #cc0000  (3 – Major)
  - Magenta:#8f1eae  (4 – Extreme)
  - Missing data: light gray #d9d9d9
